Vision-Flan: An Open-Source Multi-Modal dataset

Created by Virginia Tech's NLP Lab. Sep 13, 2023

This dataset consists of 300k examples. Each example contains an instruction, an input image, and the correlated output. Our plans for this dataset is that it can be used for training multi-modal instruction models.
